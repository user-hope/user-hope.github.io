## Hooks

Owl 提供的大多数的 hook 都与组件的生命周期相关, 但是期中一些 hook, 例如: `useComponent` 提供了一种构建特定的 hook 的方法;

使用这些 hook, 可以完成很多自定的 hook, 用来解决一些特定的问题; 下面是 odoo Web 框架提供的一些 hook;

- **useAssets**: 用来加载 assets
